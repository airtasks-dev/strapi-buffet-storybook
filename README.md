## Get Started:

1. Fork the repository
2. Clone your fork `git clone <my-fork>.git`
3. Install the dependencies `cd buffet && yarn setup`
4. Start the watch mode `yarn watch`
5. Start the documentation: `cd docs && yarn storybook`
